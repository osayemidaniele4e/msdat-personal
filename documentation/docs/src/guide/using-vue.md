# Overview of Dashboard


## App info

The system displays a modal on the application main page with a welcome message . The modal disables the main page but keeps it visible. 

- A hyperlink to watch the tutorial video is placed on the welcome page. 
- The modal also contains other pages that introduce the tabs with snapshots on the dashboard. 
- Options to skip or start a tour guide is also provided.  
- A cancel button is placed on the top right corner of the modal to allow the user view the dashboard


## Header Section 

The header tab provide links to different sub dashboard on the MSDAT platform such as
Covid-19: Service Uptake, Custom Dashboard , Other dashboards and Login/Register link are made available on the header.  When a user clicks any of the links , the system routes the page to the selected dashboard using the single page application framework.  


## Login and Sign up

The login/Register is placed at the top right corner. Users can browse through the MSDAT platform without logging in. When users click on the login/Register menu, a window pops up with a field to either log in on the register. Users with accounts have privileges to access locked features on the platform.
Register - The user can register by filling the following fields; name (first and last), email, organization, password, and confirm password. All fields are compulsory. 
Login - The user can log in with email address and password all fields are compulsory. 


## Control Panel

The panel on the dashboard allows the user to navigate around the dashboard  and take actions by selecting any of the tabs provided.  When any of the tabs is selected, the system automatically loads the page to the selected tab. 

Below is a list of tabs that can be selected on the control panel:

### Indicator Overview

This section contains the indicator table, state chart and trend analysis. 
Indicator table: This table shows the values of the selected indicator (selected from the indicator filter) across data sources.The table also shows two related indicators with their values across the data sources. Data sources are classified under three categories: i) Routine, ii) National Surveys and iii) Other Estimates 
State chart: This chart shows indicator data for the selected indicator across the states in Nigeria. The National target line is drawn to easily identify what states fall below or above the National target.
Trend analysis: This is a chart for multiple data sources showing the trend of an indicator over time.The target line can also be seen on this chart. The data sources are differentiated by colors which can be identified using the legends displayed above the lines.
