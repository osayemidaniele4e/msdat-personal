# Documentation for CONFIG FOLDER 


>This folder holds files for configuration settings for IndexxedDB (Dexie) and Axios

## Files in this folder

1. api.js
2. dexie.js
3. endpoint.js

### api.js

> This file instantiates an axios instance using just one property, the baseUrl, gotten from the enviroment variables

### dexie.js

> This file instantiates and exports the indexeddb database instance using dexie. It sets the name of the database as well as the object stores.

### endpoint.js

> Thus file exports an object containing names of all the endpoints used by the dashboards, conveniently named for the purpose of easy referencing.
